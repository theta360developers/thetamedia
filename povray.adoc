== Povray

This shows image adjustment with Povray.

=== .pov configuration file

  #include "colors.inc"
  #include "shapes.inc"

  camera {
  spherical
  }

  sphere {
    // center of sphere
    <0,0,0>, 1
    texture {
      pigment {
        image_map {
          jpeg "museum.jpg"
          interpolate 2
          once
          map_type 1

        }
       }
       rotate z * 90
       // rotate x * 90
       finish { ambient 1}
    }
    }

=== command line

  $ povray +W5376 +H2688 +fj  thetasphere.pov +Othetaspherez90.jpg
